# Legal-Document-Analyser
It is gonna to analyze the document and then provide summary in a manual language.

#Tools used in Analyzer
1. EasyOCR
2. Tranformer Pipeline
3. Transformer = LegalPegasus used to analyze the legal document (Pegasus is a Google Transformer Trained on Legal dataset to analyze the Leagal documents).

#Steps To Convert any Legal document into a manual summary
1. EasyOCR extract the text from the document
2. Then , LeaglPegasus Tranformer analyse the text
3. And then generate the summary .
